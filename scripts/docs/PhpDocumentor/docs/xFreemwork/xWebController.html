add_meta (line 39)

Convenience method for merging this instance metadata with additional metadata.

The given metadata will erase this instance metadata. The merged metadata array replaces this instance metadata and is also returned. Metadata name format can be:

  • plain: eg. somename
  • hirearchic: eg. some/meta/name, which will result in an array('some' => array('meta' => array('name' => $contents)))

  • return: Merged metadata array.
array add_meta (array|string $meta, [mixed $contents = null])

  • mixed $contents: (Mandatory if 1st param is a string) The metadata contents.

session (line 67)

Used to save/retrieve value in controller session.

If only key param given, the corresponding stored value is returned (null if key does not exist) If both key and value are given, the value is stored in session. If no argument given, return the session array of the controller.
