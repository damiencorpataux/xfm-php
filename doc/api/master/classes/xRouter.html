add(string $pattern, $params) 

Parameters

$pattern

string

A rails-like route pattern, made of: - static keywords (e.g. somekeyword) - variable parameters, those will be set as parameters assigned with the route fragment value when calling the controller (e.g. :aparameter) Example: /somekeyword/:controller/:action/:page
