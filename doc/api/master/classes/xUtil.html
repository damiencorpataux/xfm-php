Merges arrays recursively, replacing existing keys.

array_merge() : array
Static

Priority is given to the last given array parameter.

Careful: This algorythm differs from PHP array_merge_recursive in the sense that merging array with same-level-same-keyname will remain unique when using xUtil::array_merge().

Merge example:

print_r(xUtil::array_merge(array(
    'Index content',
    'assoc' => array(
        'a_key' => 'a_content',
        'b_key' => 'b_content',
        '2nd level index content'
    ),
    array(
        'key' => 'value'
    ),
    'this-key' => 'remains-unique',
), array(
    'assoc' => array(
        'a_key' => 'a_content_modified',
        'c_key' => 'c_content',
    ),
    array(
        'key' => 'value'
    ),
    'Index content 2',
    'this-key' => 'remains-unique'
)));

will output:

Array
(
    [0] => Index content
    [assoc] => Array
        (
            [a_key] => a_content_modified
            [b_key] => b_content
            [0] => 2nd level index content
            [c_key] => c_content
        )

    [1] => Array
        (
            [key] => value
        )

    [2] => Array
        (
            [key] => value
        )

   [3] => Index content 2
   [this-key] => remains-unique
)
